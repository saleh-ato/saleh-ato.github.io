خدمات

من طیف وسیعی از خدمات را ارائه می دهم، با تخصص در زمینه های مختلف می توانم با استفاده از فریم ورک Flask به شما در طراحی وب و API کمک کنم. علاوه بر این، خدمات طراحی وب با استفاده از وردپرس، طراحی گرافیک با ابزارهای استاندارد صنعتی مانند فتوشاپ و ایلوستریتور، ویرایش ویدیوی جزئی، عکاسی و تولید محتوای سئو را ارائه می دهم.

طراحی وب

طراحی وب با استفاده از پایتون (فریم‌ورک Flask)، بوت‌استرپ، جاوااسکریپت، جی‌کوئری، HTML، CSS و طراحی وب با وردپرس

طراحی گرافیک

طراحی پست‌های اینستاگرام، بنر، استوری، کارت ویزیت، بنرهای چاپی، فلایر، بروشور، الگوهای پارچه و غیره

مدیریت سرور

به‌روزرسانی پکیج‌ها، اجرای اسکریپت‌ها روی وب‌سرور و غیره

سئو و عکاسی محصول

با تمایل زیاد به یادگیری و رشد، از طریق درک اساسی از مفاهیم سئو، به دنبال بهینه‌سازی وب‌سایت‌ها و ارائه حداکثر دید و ارزش به صاحبان کسب و کار هستم.
